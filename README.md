- Hướng dẫn sử dụng:


B1: Copy Url của repository này

B2: Tạo thư mục "XXX" để chuẩn bị chứa repository

B3: truy cập CMD trong "XXX" và gõ lệnh "git clone <Url của repository này>"

B4: Mở Visual Studio và chọn thư mục vừa CLONE về

B5: Vào Tab Tools -> Nuget Package Manager -> Package Manager Console

B6: bên trong Console, ghi "cd BaiThucHanh01_Nhom2_DinhDuy23810310435" 

B7: viết lần lượt "dotnet ef migrations add PostGreSQLDataBase" + "dotnet ef database update"

B8: Vào phần mềm PostGreSQL để kiểm tra Database đã được cập nhật chưa?

B9: Vào lại Visual Studio dự án đang mở, nhấn nút "f5" hoặc click vào nút "Run"

B10: Tận hưởng sản phẩm \(O.O)/
